## EMERGENCE SIM v2.0 — ZERO PRE-SCRIPT

Solo le **leggi fisiche** sono codificate. Lingua, strumenti, civiltà e terraformation devono **emergere** — mai missioni scriptate.

Manifesto: **[`docs/EMERGENCE-SIM-v2.md`](docs/EMERGENCE-SIM-v2.md)**

| Livello | Contenuto |
|---------|-----------|
| **L0** Fisica | Termo, gravità, idrologia, erosione |
| **L1** Mondo | Genesis, clima, biomi, risorse |
| **L2** Biologia | DNA 256-D, metabolismo, selezione |
| **L3** Cognizione | Percezione locale, plasticità (NEAT) |
| **L4** Civiltà | Commercio, costruzione emergente, politica, parola |

**Osservatore:** `make earth-console` → http://127.0.0.1:8090/

## Stato del progetto

| Area | Stato | Dettaglio |
|------|-------|-----------|
| Fasi 0–2 | ✅ | Cognizione, riproduzione, lessico |
| Fase 4 | ✅ | Agricoltura, scrittura, politica, metallurgia |
| Waves 16–41 | ✅ | Genesis → clima → insediamenti → render |
| **Realismo Terra (globale)** | **~76 %** | Media 7 dimensioni → [`docs/ROADMAP-REALISME-TERRE.md`](docs/ROADMAP-REALISME-TERRE.md) (obiettivo **80 %**) |

**Test:** **133** pytest · smokes **p72–p86**

> **Nota:** Vecchi documenti indicavano **68 %**, **74 %** o **80 %** come globale. **~76 %** è la media unificata.
