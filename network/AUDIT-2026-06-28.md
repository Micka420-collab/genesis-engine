1. **Rate-limit par IP** (fenêtre glissante, `RATE_MAX_PER_WINDOW`/min, X-Forwarded-For pris en compte pour nginx/Cloudflare) sur `register`/`work`/`submit` → HTTP 429. Test `test_rate_limit_per_ip`.
