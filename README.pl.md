## EMERGENCE SIM v2.0 — ZERO PRE-SCRIPT

Tylko **prawa fizyki** są zakodowane na stałe. Język, narzędzia, cywilizacja i terraforming muszą **emergować** — bez scenariuszowych questów.

Manifest: **[`docs/EMERGENCE-SIM-v2.md`](docs/EMERGENCE-SIM-v2.md)**

| Warstwa | Zawartość |
|---------|-----------|
| **L0** Fizyka | Termo, grawitacja, hydrologia, erozja |
| **L1** Świat | Genesis, klimat, biomy, zasoby |
| **L2** Biologia | DNA 256-D, metabolizm, selekcja |
| **L3** Poznanie | Lokalna percepcja, plastyczność (NEAT) |
| **L4** Cywilizacja | Handel, emergentna budowa, polityka, mowa |

**Obserwator:** `make earth-console` → http://127.0.0.1:8090/
