## Contexte

Wave 1 (session 9) avait livré les 4 modules de connaissance : `physics`,
`chemistry`, `material_synthesis`, `statics`. Elle traitait le bronze
binaire (Cu+Sn) avec une règle de mélange linéaire (Vegard) sur la
dureté.

Wave 2 demande de modéliser :

1. **Alliages ternaires** — au moins 3 éléments dans la composition.
2. **Dopage non-linéaire** — l'ajout de 1-2 % d'un élément (C dans Fe →
   acier) doit produire un effet **non-linéaire** sur la dureté
   (lattice distortion / dislocation pinning).
3. **Registre par culture** — chaque culture connaît un sous-ensemble
   distinct des matériaux découverts.
4. **Transmission de recettes** — une culture peut enseigner à une
   autre, sous contrainte de probabilité (mémoire orale).

L'API ``synthesize(composition, conditions, tools, culture_id, ...)``
supportait déjà des compositions dict-de-n-éléments. Les **points 3 et 4**
étaient déjà implémentés dans `MaterialRegistry.register`,
`culture_known`, `transmit`. **Le seul vrai travail neuf est le
mécanisme de dopage non-linéaire**, plus un smoke d'intégration qui
valide tout le pipeline bout-en-bout.

## Livrables

### Dopage non-linéaire (`engine/material_synthesis.py`)

Deux helpers + une modification de `_derive_properties` :

```python
_DOPING_HOST_THRESHOLD = 0.80   # un seul élément >= 80 % qualifie d'hôte
_DOPING_MAX_DOPANT_FRAC = 0.10  # 0 < fraction < 10 % = dopant
_INTERSTITIAL_DOPANTS = {"C", "N", "B", "H"}

def _detect_doping(composition):
    """Retourne (host_symbol, [(dopant, fraction), ...]) si la composition
    matche un pattern dopage, sinon (None, [])."""

def _doping_hardness_boost(host, dopants):
    """Boost non-linéaire en Mohs.
       6× base pour les dopants interstitiels (C, N, B, H)
       2× base pour les substitutionnels (Mn, P, Sn, ...)
       Sous-linéaire : sqrt(min(20×fraction, 1.0)) — saturation à ~5 %.
       Cap à +5 Mohs."""
```

`_derive_properties` calcule maintenant `hardness_linear` (Wave 1, inchangé)
puis ajoute le boost de dopage si la composition matche. Trois nouveaux
champs apparaissent dans `properties` :

- `hardness_linear_only` — la valeur Wave 1 stricte.
- `doping_boost_mohs` — le delta apporté par le pattern dopant.
- `is_doped` — 1.0 si pattern dopage, 0.0 sinon.

### Calibration empirique du dopage

| Matériau | linear | boost | total | réel (Mohs) |
|---|---|---|---|---|
| Fe pur | 1.79 | 0.00 | 1.79 | 4 (calibration globale conservatrice) |
| **Acier** (Fe + 1.5 % C + 1.5 % Mn) | 1.79 | **+4.38** | **6.17** | 4-7 ✓ |
| Bronze (Cu + 30 % Sn) | 1.79 | 0.00 | 1.79 | 3 (binaire ≠ dopage) |
| **Phosphor bronze** (Cu + 5 % Sn + 1 % P) | 1.79 | **+2.89** | **4.68** | 3-4 ✓ |

L'écart Fe-pur → acier (+4.4 Mohs) reproduit qualitativement la
différence entre fer doux et acier martensitique.

## Pourquoi le dopage est un *vrai* Wave 2

Wave 1 modélisait le bronze binaire avec une règle linéaire — l'acier
binaire Fe+C lui donnerait `hardness ≈ 0.97 × Mohs_Fe + 0.03 × Mohs_C ≈
2`, soit *plus mou* que l'acier réel. Wave 2 capture l'effet physique
critique : les dopants minoritaires interstitiels distordent la
maille cristalline, épinglent les dislocations, et durcissent
**non-linéairement** le matériau.

Le test step 3 démontre la qualité du pattern : un binaire 70/30
(bronze) n'est *pas* tagué doped — c'est une vraie solution solide,
pas un dopage. Un binaire 94/5/1 (phosphor bronze) l'est. La règle
discrimine les deux régimes physiques distincts.

## Conséquences pour la vision

> *"Avec les mêmes briques de Lego que nous, qu'auraient-elles pu
>  construire d'autre ?"* — FUTURE-VISION.md

Avec le dopage Wave 2 disponible aux agents :

- Une civilisation qui découvre que **0.5-2 % de C dans Fe** durcit
  spectaculairement → âge du fer puis acier rapidement.
- Une autre qui privilégie le **Cu + B/N** (jamais essayé sérieusement
  par notre histoire) peut produire un alliage Cu-N à dureté
  comparable. → divergence civilisationnelle exploitable.
- L'algorithme de `_detect_doping` est **agnostique** des recettes
  humaines connues : il valide n'importe quel pattern host+dopant.

Wave 3 (à venir) : composites bois-céramique, matrices fibre-renforcée,
chimie hors lithosphère (atmosphères CO2-rich, températures extrêmes).
