```

  MATE      1883      ← preempts SPEAK quand fertile (cognition.decide ordering)
  IDLE       782
  SPEAK      543      ← se déclenche dès que mating cooldown actif

  WALK_TO    208
  FORAGE     144      ← matériaux + nourriture
  DRINK      111      ← survie OK
  EAT         51
```

**Conclusion comportementale :** la cognition Phase 4 priorise mating → quand mating impossible (drive trop haut, partenaire infertile), la suite (SPEAK, SHARE, EXPLORE) se libère. Avec drive_accel=1500 l'horloge biologique tourne 1500× plus vite que le temps de simulation, ce qui sature les drives — c'est ce qui empêche les `build` complets en 1000 ticks (hearth = 2h de labor, mais agents toujours en thirst > 0.65 → SPEAK gate fermé épisodiquement).
