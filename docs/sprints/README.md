## 🔄 Naming convention

`YYYY-MM-DD_<TYPE>-<topic>.md` where `<TYPE>` is one of:

- `PHASE<n>-PROGRESS` — multi-day phase progress snapshot
- `PHASE<n><letter>-PLAN` / `-STATUS` — planning / status of a sub-phase
- `SPRINT` — single-day session summary
- `AUDIT-REPORT` — mid-project gap analysis
- `WORLD-VEILLE-REPORT` — technology watch on the industry

This convention is enforced going forward. Old files without dates were renamed accordingly.
